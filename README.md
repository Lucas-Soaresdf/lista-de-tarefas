# Lista 

Projeto de lista de tarefas desenvolvido com HTML, CSS e JavaScript com foco na prática de lógica de programação, manipulação do DOM e responsividade.

A aplicação permite adicionar, editar e remover tarefas dinamicamente, além de contar com uma interface adaptada para diferentes tamanhos de tela. Durante o desenvolvimento pratiquei eventos, arrays, funções e organização do código.

## Funcionalidades

* Adicionar tarefas
* Editar tarefas
* Remover tarefas
* Limpar lista
* Interface responsiva

## Linguagens

* HTML5
* CSS3
* JavaScript
